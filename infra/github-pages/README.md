# GitHub Pages Hosting

`legacy-v1` may continue to be used for GitHub Pages while the 2026 portfolio app is being built.

The long-term hosting decision is still open:

- Keep `legacy-v1` on GitHub Pages until `apps/portfolio-web` is ready.
- Archive `legacy-v1` after the new app becomes production.
- Use GitHub Pages only for static legacy hosting and move the new app to Vercel.

No production deployment secrets should be committed to this repository.
